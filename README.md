# my-ai-database

# AI OS

A personal knowledge system for capturing, organizing, and growing knowledge over time.

---

## Purpose

This repository is the permanent knowledge base of my AI OS.

The goal is not to collect everything, but to build reusable knowledge that compounds over time.

---

## Repository Structure

```
my-ai-database/
├── captures/
├── knowledge/
├── projects/
├── docs/
└── README.md
```

- **captures/** : Raw ideas and daily captures
- **knowledge/** : Long-term reusable knowledge
- **projects/** : Active projects
- **docs/** : AI OS documentation and rules
