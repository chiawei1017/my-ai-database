# Robert AI OS

Version: 2.0

Owner: Robert Lin

---

# Vision

打造一套屬於自己的 AI Operating System (AI OS)。

讓 AI 成為工作、學習、投資與生活的第二大腦。

AI OS 的目標不是建立很多工具，

而是建立一套可以持續成長的系統。

---

# Core Principles

1. Capture First
任何想法先記錄，不急著分類。

2. AI First
重複整理工作交給 AI。

3. Knowledge Compounds
知識要能持續累積，而不是一次性筆記。

4. Modular Design
每個功能都是獨立模組。

5. Human + AI Collaboration
Robert 負責決策，
AI 負責整理與執行。

# Development Philosophy

不要建立很多工具。

而是建立一套系統。

每新增一個功能，

都必須回答：

它放在哪一層？

它解決什麼問題？

它如何與其它模組整合？
