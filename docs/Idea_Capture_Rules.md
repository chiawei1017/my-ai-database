# Idea Capture Rules

## Principle

不要一開始就判斷資訊價值，先確保不要失去它。

## Quick Rules

- 需要處理 → Inbox
- 只是有趣 → Spark
- 未來可能變專案 → Seeds
- 已整理且有長期價值 → Knowledge Center
- 工作相關 → Work Center
- 投資相關 → Investment Center
- 生活或學習相關 → Personal Center
- 想實驗或開發 → AI Lab

## Daily Workflow

1. 白天快速記錄到 Inbox、Spark 或 Seeds。
2. 不急著整理。
3. 每天或每週請 AI Processor 協助分類。
4. 有行動價值的轉成 Todo。
5. 有長期價值的轉進 Knowledge、Work、Investment 或 Personal。

## Examples

### Meeting note

放入 Inbox。

### 有趣但不確定有沒有用的想法

放入 Spark。

### 桌遊開發靈感

先放入 Seeds，未來正式開始時再移到 AI Lab。

### SOP 或讀書筆記

整理後放入 Knowledge Center。
